This directory uses an explicit taxonomy so contributors do not have to grep the repo like animals every time they need current truth.

## Canonical Paths

- Current state: [`CURRENT_STATE.md`](CURRENT_STATE.md)
- Runbooks: [`runbooks/README.md`](runbooks/README.md)
- ADRs: [`adr/README.md`](adr/README.md)
- Historical context: [`history/README.md`](history/README.md)

## Rules

- `CURRENT_STATE.md` is the single canonical markdown source for current backend/runtime state.
- `README.md` at repo root stays short and points here.
- Runbooks describe procedures.
- ADRs capture decisions.
- Historical docs keep context but are not current truth.
